# 隔离策略对比实验结果

## 实验配置

- 总内存限制: 2560MB
- 回放速度: 100x

## 策略对比

| 策略 | HIGH 时间 | LOW 平均 | 优先级比 | OOM | memory.high 事件 | BPF 触发 |
|------|----------|----------|---------|-----|-----------------|---------|
| 无隔离 | 4.5s | 1.8s | 0.87x | 0 | 0 | 0 |
| 静态限制 | 1.1s | 1.3s | 1.08x | 0 | 0 | 0 |
| BPF 动态 | 6.5s | 79.6s | 7.40x | 0 | 19578 | 1556 |

## 分析

### BPF vs 无隔离

- **优先级比改善**: 0.87x -> 7.40x (+748.4%)
- **BPF 延迟触发次数**: 1556

### BPF vs 静态限制

- **OOM 事件**: 0 -> 0
- **优先级比**: 1.08x -> 7.40x

## 结论

BPF 动态优先级隔离相比无隔离策略，优先级比改善了 **748.4%**，
有效保护了 HIGH 优先级任务的完成时间。
